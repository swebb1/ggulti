# Convenience function to create pitch_arrows which trace an objects movement path on the pitch e.g. a players cut. You can choose to only trace specific object types e.g offense players only.

Convenience function to create pitch_arrows which trace an objects
movement path on the pitch e.g. a players cut. You can choose to only
trace specific object types e.g offense players only.

## Usage

``` r
object_paths(object_list, objects = c("Offense"))
```

## Arguments

- object_list:

  A list of object data.frames e.g. generated by pitch_object (required)

- objects:

  A vector of the object types to trace : default = "Offense"

## Value

A data.frame

## Examples

``` r
object_paths(player_list)
#> Error: object 'player_list' not found
object_paths(player_list, c("Offense","Defense"))
#> Error: object 'player_list' not found
```
