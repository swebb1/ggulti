# Export an object or arrow list as a tsv file arranged by frames. This is useful for making manual edits.

Export an object or arrow list as a tsv file arranged by frames. This is
useful for making manual edits.

## Usage

``` r
export_frames(list, filename = "frames.tsv")
```

## Arguments

- list:

  Either a list of objects or arrows

- filename:

  Name of the file to save : default = "frames.tsv"

## Value

A data.frame

## Examples

``` r
export_frames(object_list)
#> Error: object 'object_list' not found
```
