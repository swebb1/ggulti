# Helper function used by plot_play() function. Takes a dataframe of objects/arrows and fills so that each object/arrow appears in all frames

Helper function used by plot_play() function. Takes a dataframe of
objects/arrows and fills so that each object/arrow appears in all frames

## Usage

``` r
all_frames(df, frames)
```

## Arguments

- df:

  Dataframe of objects/arrows

- frames:

  Total number of frames to fill

## Value

A data.frame

## Examples

``` r
all_frames(throws,3)
#> Error: object 'throws' not found
```
